# Valiform
Validate any form automagically with native HTML5 tags and properties


## Code inclusion
```html
<script
    src="Valiform.js"
    charset="utf-8"
    override_html5_validation="true"
    debug="false">
</script>

<!-- OPTIONAL VALIFORM LOCALIZED MESSAGES -->
<script src="Valiform_messages.js" lang="en"></script>

```
